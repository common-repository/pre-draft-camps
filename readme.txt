== Description ==

This plugin create Pre Draft Camp environment on your website.  
  
You can create/manage Camp, Venue, Coach and Players within your website.  
Player can register for any camp by filling form and do payment of camp fees mentioned in admin panel.  
I used Paypa PRO for payment and admn can change the credentials from admin panel under Settings Tab.  
  
Shortcodes are available for upcoming camp listing and detail page for any camp by id and camp listing.  
  
Upcoming camp listing: [upcoming_camps]  
  
Camp Detail Page: [pdc_camp campid=CAMPID]  
  
Camp Listing Page: [pdc\_camp\_listing]  
  
Plugin contain its own template for camp detail page.  
  
Widget is also here to show up-coming camp in slide show way.  
  
I have tested it and 100% working.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
